# Digital Literacy - Evaluated Course Project

## About This Project
This repository is my submission for the CSE0001 – Digital Literacy course at VIT Bhopal University. The project is structured around five tasks that cover different aspects of being digitally literate, from building an online presence to staying safe on the internet. I put this together as part of the Student Digital Ambassador scenario described in our course brief.

## Module Summaries

### Task 1 – Digital Literacy Infographic
Created a one-page infographic using Canva that explains what digital literacy is, lists useful tools for students, and covers safe internet practices.

### Task 2 – Student Digital Portfolio
Set up profiles on GitHub, LinkedIn, and Kaggle. Built a GitHub profile README with my details and learning goals.

### Task 3 – Coding & Collaboration Platforms
Solved a beginner challenge on HackerRank and created a 5-question Digital Literacy Awareness Quiz using Google Forms.

### Task 4 – Email Etiquette & Online Communication
Drafted two professional emails (one to a professor, one to an internship coordinator) and made a social media do's and don'ts checklist.

### Task 5 – Cybercrime Awareness
Wrote a case study on UPI/Online Payment Fraud and prepared an 8-point prevention checklist for college students.
